Compte rendu du projet ASPE 2018-2019

Sujet du projet Web

Nous avons décidé de développé un site orienté sur la vente de voyage. Vous trouverez sur notre site plusieurs propositions de voyage.

Contraintes et problèmes rencontrés

Voici un liste des problèmes que nous avons rencontrés lors de la création du site:

      Le problème du responsive design:
      En effet il fallait que notre site web puissent s'adapter à la taille de l'écran. Nous aurions aimé utilisé le framework Bootstraps car il contient de nombreux composants prêts à l'emploi (bouton, menu, liste déroulantes etc...).Cependant son utilisation allait à l'encontre des consignes. Nous nous sommes ensuite tourné vers la spécification CSS3 Media Queries. Cependant par manque de temps, nous n'avons pu développer une fiche css prenons en compte la taille des écrans.
      L'impossibilité de rendre le site dynamique:
      Nous ne pouvions pas utiliser de JavaScript pour la création de notre site. JQuery. La bibliothèque contient notamment les fonctionnalités suivantes :
    1. Parcours et modification du DOM (y compris le support des sélecteurs CSS 1 à 3 et un support basique de XPath) ;
    2. Evénements ;
    3. Effets visuels et animations ;
    4. Manipulations des feuilles de style en cascade (ajout/suppression des classes, d'attributs…) ;
    5. Ajax ;
    6. Plugins ;
    7. Utilitaires (version du navigateur web…).
